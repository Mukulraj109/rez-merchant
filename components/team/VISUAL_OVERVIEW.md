## Responsive Behavior

### Mobile (< 768px)
- Full width cards
- Stacked buttons
- Single column layout
- Collapsed categories by default

### Tablet (768px - 1024px)
- 2 column grid for cards
- Side-by-side buttons
- Expanded modal width

### Desktop (> 1024px)
- 3 column grid for cards
- Full width permission matrix
- Split view for details
